# Documentation Index

This folder is the primary documentation for the `jff-arch-config` platform.

## Start Here
- `docs/bootstrap.md`: first-time setup, install prerequisites, and first run.
- `docs/architecture.md`: project structure, data model, and execution model.

## Daily Usage
- `docs/cli.md`: CLI command reference and practical workflows.
- `docs/tui.md`: TUI dashboard usage and controls.
- `docs/services.md`: systemd user unit strategy and service operations.
- `docs/cheatsheet.md`: one-page quick reference.
- `docs/troubleshooting.md`: common errors and fixes.

## Safety and Recovery
- `docs/recovery.md`: rollback and incident response playbook.

## Recommended Learning Order
1. Read `docs/bootstrap.md`.
2. Run `doctor`, then `quickstart` from `docs/cli.md`.
3. Use `docs/tui.md` for interactive operation.
4. Configure timers/services from `docs/services.md`.
5. Keep `docs/recovery.md` open before applying risky changes.
