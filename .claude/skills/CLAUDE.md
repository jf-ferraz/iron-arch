# Skills

| Skill | When to Load |
|-------|-------------|
| `planning/SKILL.md` | Complex multi-step work requiring task breakdown and dependency tracking |
| `debugging/SKILL.md` | Hard-to-diagnose problems, intermittent failures, unclear root causes |
| `refactoring/SKILL.md` | Code improvement without behavior change — priority classification, safe techniques |
| `quality-review/SKILL.md` | Deep code quality analysis organized by cognitive mode |

Skills are on-demand reference documents. Load a skill when you need deeper guidance on a topic than the agent prompt provides. Skills don't replace agents — they augment them with detailed methodology.
