# Conventions

| Convention | Scope |
|-----------|-------|
| `code-quality.md` | Universal code quality standards — structure, naming, complexity thresholds |
| `temporal.md` | Temporal contamination detection — 5-question heuristic for clean comments |
| `documentation.md` | Documentation hierarchy, invisible knowledge, CLAUDE.md conventions |
| `git-discipline.md` | Commit practices, branch naming, known-good increments |
| `severity.md` | Issue classification: MUST / SHOULD / COULD — with intent markers |

Conventions are role-independent rules. Every agent respects them. They're separate from agent prompts so updating a standard doesn't require editing 6 files.
